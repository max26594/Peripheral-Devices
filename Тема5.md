### 2. Характеристики SSD (Типи флешпам'яті NAND)

<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/858c1f90-6052-4492-b5c5-9dbac0158b76" />

 Тип комірки (Flash) | Опис та Характеристики |
| :--- | :--- |
| **SLC (Single-Level Cell)** | 1 біт на комірку. Найвища швидкість та довговічність, але водночас і найвища вартість. |
| **MLC (Multi-Level Cell)** | 2 біти на комірку. Хороший баланс між продуктивністю та витривалістю. |
| **TLC (Triple-Level Cell)** | 3 біти на комірку. Стандарт для більшості сучасних ноутбуків та ігрових ПК, пропонує оптимальне співвідношення швидкості та вартості. |
| **QLC (Quad-Level Cell)** | 4 біти на комірку. Найвища ємність за найнижчої ціни, проте має найнижчу швидкість та витривалість. |

## Cкрипт чи програма для відображення технічної інформації про встановлений на ПК накопичувач

### Зображення
<img width="983" height="772" alt="image" src="https://github.com/user-attachments/assets/e148aee5-0b98-4091-8a72-ceaf6b7fe49f" />

### Cкрипт 

```batch
@echo off
chcp 65001 >nul

echo ---------------------------------------
echo         Iнформацiя про диски
echo ---------------------------------------
echo.
echo [ФIЗИЧНI НАКОПИЧУВАЧI (Залiзо)]:
powershell -NoProfile -Command "Get-PhysicalDisk | Select-Object @{Name='Model';Expression={$_.FriendlyName}}, MediaType, BusType, HealthStatus, @{Name='Total (GB)';Expression={[math]::Round($_.Size / 1GB, 2)}} | Format-Table -AutoSize"

echo [ДЕТАЛЬНИЙ СТАН SSD (Темп, Знос, Години роботи, Помилки)]:
powershell -NoProfile -Command "Get-PhysicalDisk | Get-StorageReliabilityCounter | Select-Object @{Name='ID';Expression={$_.DeviceId}}, @{Name='Temp (C)';Expression={$_.Temperature}}, @{Name='Wear (%%)';Expression={$_.Wear}}, @{Name='PowerOn Hours';Expression={$_.PowerOnHours}}, @{Name='Read Errors';Expression={$_.ReadErrorsTotal}} | Format-Table -AutoSize"

echo [ЛОГIЧНI РОЗДIЛИ, РОЗМIР ТА ВIЛЬНЕ МIСЦЕ]:
powershell -NoProfile -Command "Get-CimInstance Win32_LogicalDisk -Filter 'DriveType=3' | Select-Object @{Name='Drive';Expression={$_.DeviceID}}, @{Name='Label';Expression={$_.VolumeName}}, @{Name='Format';Expression={$_.FileSystem}}, @{Name='Total (GB)';Expression={[math]::Round($_.Size / 1GB, 2)}}, @{Name='Free (GB)';Expression={[math]::Round($_.FreeSpace / 1GB, 2)}} | Format-Table -AutoSize"

echo [ПРИХОВАНI СИСТЕМНI ФАЙЛИ НА C: (Гiбернацiя та Пiдкачка)]:
powershell -NoProfile -Command "Get-Item C:\pagefile.sys, C:\hiberfil.sys, C:\swapfile.sys -ErrorAction SilentlyContinue | Select-Object Name, @{Name='Size (GB)';Expression={[math]::Round($_.Length / 1GB, 2)}} | Format-Table -AutoSize"

echo [СТАТУС TRIM (Для довговiчностi SSD)]:
echo (Якщо DisableDeleteNotify = 0, то TRIM увiмкнено - це ДОБРЕ)
fsutil behavior query DisableDeleteNotify
echo.

echo [ПЕРЕВIРКА ЦIЛIСНОСТI ФАЙЛОВОЇ СИСТЕМИ C: (Dirty Bit)]:
fsutil dirty query c:
echo.

echo ---------------------------------------
echo    Тестування швидкостi диску C:
echo ---------------------------------------
echo Йде перевiрка (це може зайняти до хвилини)...
echo.

winsat disk -drive c | findstr /I /C:"Disk" /C:"Average" /C:"Latency"

echo.
echo ---------------------------------------
echo Тестування завершено
pause
